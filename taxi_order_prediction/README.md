### Проект по прогнозированию количества заказов такси на следующий час, временные ряды
- `Python 3` `Jupyter Notebook`
- `pandas`, `matplotlib`, `seaborn`, `numpy`, `scipy`, `phik`, `scikit-learn`, `statsmodels`
- Данные предоставлены Яндекс Практикумом 
- Проанализировали данные, визуально на графиках заметили наличие тренда и 2-х типов сезонности
- Провели статистический тест на стационарность
- Проверили автокорреляцию. Убедились в наличии сезонностей. Посмотрели силу влияния разных лагов. Для обучения отделили значимую часть ряда
- Обучили 3 разные модели (линейную регрессию, дерево решений и модель для временных рядов sarimax) c различными гиперпараметрами
- Выбрали лучшую модель (линейную регрессию) и проверили ее качество на тестовой выборке rmse = 31.98, что удовлетворяет заданным критериям качества
- `Бизнес-результат`: с помощью прогнозов модели компания может ругулировать количество такси у аэропорта на следующий час, что позволяет добится наилучшего соответствия спроса и предложения
