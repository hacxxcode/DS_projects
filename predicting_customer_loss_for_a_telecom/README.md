### Проект по прогнозированию оттока клиентов для телекоммуникационной компании, классификация
- `Python 3` `Jupyter Notebook`
- `pandas`, `matplotlib`, `seaborn`, `numpy`, `scipy`, `phik`, `scikit-learn`, `lightgbm`, `catboost`
- Данные предоставлены Яндекс Практикумом и состоят из нескольких файлов, полученных из разных источников, включают данные клиентов и детализацию по договорам
- Данные были предобработаны, выделен target, проведен исследовательский и корреляционный анализ с визуализацией результатов, feature engineering
- На кросс-валидация с подбором гиперпараметров обучено 4 модели
- Лучшая модель (градиентный бустинг) на тестовой выборке превзошла требуемый порог 
- Проведен анализ лучшей модели, сделаны выводы и рекомендации
- Бизнес-результат: с помощью прогнозов модели оптимизируются расходы компании на удержание клиентов
